# KaBang
1. 난이도 : 중/하

- 실행 : java -jar KakaoBank.jar {input file} {template file}
    - example : java -jar KakaoBank.jar input.txt template.txt

- parameter :
    - input file : data file ( default : input.txt )
    - template file : template file ( default : template.txt )
    
- result : output.txt
